### pa_averages_report.txt

This report contains information about the average number of districts won by democrats in the
2016 Presidential and Senatorial elecitions. For each method, there is an average for all the
plans in a particular ensemble file, and then there is a final value reporting the average
over all plans in all ensembles for that method.
